жҢ‘жҲҳеңЁдәҺпјҡLLM rollout жҳҜдёҖж¬ЎжҖ§иҮӘеӣһеҪ’з”ҹжҲҗпјҢиҖҢ VLA rollout йңҖиҰҒдёҺзү©зҗҶ/д»ҝзңҹзҺҜеўғиҝӣиЎҢй—ӯзҺҜеӨҡиҪ®дәӨдә’вҖ”вҖ”жҜҸжӯҘжү§иЎҢеҗҺзҺҜеўғзҠ¶жҖҒж”№еҸҳпјҢжЁЎеһӢеҝ…йЎ»еҹәдәҺе®һж—¶ж„ҹзҹҘеҸҚйҰҲз”ҹжҲҗдёӢдёҖжӯҘеҠЁдҪңгҖӮиҝҷиҝңжҜ” LLM RL жӣҙж…ўгҖҒжӣҙжҳӮиҙөгҖӮ

еӣҫ1пјҡSimpleVLA-RL жЎҶжһ¶жҖ»и§ҲгҖӮиҜҘеӣҫд»Ҙеӣӣж ҸеёғеұҖеұ•зӨәдәҶ SimpleVLA-RL зҡ„ж ёеҝғиҙЎзҢ®пјҡ(1) RL и®ӯз»ғжөҒзЁӢвҖ”вҖ”д»Һ SFT еҲқе§ӢеҢ–зҡ„ VLA жЁЎеһӢеҮәеҸ‘пјҢйҖҡиҝҮ GRPO еңЁзәҝ RL жҢҒз»ӯж”№иҝӣзӯ–з•ҘпјҢеҲ©з”ЁдәҢе…ғ outcome rewardпјҲжҲҗеҠҹ=1пјҢеӨұиҙҘ=0пјүй©ұеҠЁеӯҰд№ пјӣ(2) жҖ§иғҪеҜ№жҜ”вҖ”вҖ”еңЁ LIBERO е’Ң RoboTwin еҹәеҮҶдёҠеӨ§е№…и¶…и¶Ҡ SFT еҹәзәҝпјӣ(3) "pushcut" зҺ°иұЎвҖ”вҖ”RL и®ӯз»ғдёӯиҮӘдё»ж¶ҢзҺ°еҮәжј”зӨәж•°жҚ®дёӯдёҚеӯҳеңЁзҡ„ж–°еҠЁдҪңзӯ–з•ҘпјҲжҺЁжҢӨиҖҢйқһжҠ“еҸ–пјүпјӣ(4) жіӣеҢ–иғҪеҠӣвҖ”вҖ”еңЁз©әй—ҙй…ҚзҪ®гҖҒзү©дҪ“зұ»еһӢгҖҒд»»еҠЎзӣ®ж ҮдёүдёӘз»ҙеәҰдёҠеқҮдјҳдәҺ SFTгҖӮиҜҘеӣҫзӣҙи§Ӯеұ•зӨәдәҶд»Һ"жЁЎд»ҝеӯҰд№ "еҲ°"ејәеҢ–еӯҰд№ "зҡ„иҢғејҸиҪ¬еҸҳеёҰжқҘзҡ„еӨҡж–№йқўж”¶зӣҠгҖӮ

дәҢгҖҒйў„еӨҮзҹҘиҜҶпјҡLLM RL дёҺ VLA RL зҡ„еҪўејҸеҢ–еҜ№жҜ”

и®әж–Үз¬¬ 2 иҠӮжҸҗдҫӣдәҶдёҖдёӘйқһеёёжё…жҷ°зҡ„ LLM RL е’Ң VLA RL еҪўејҸеҢ–еҜ№жҜ”гҖӮиҝҷдёҖиҠӮжҳҜзҗҶи§Јж•ҙзҜҮе·ҘдҪңзҡ„е…ій”®еҹәзЎҖвҖ”вҖ”еӣ дёәе°Ҷ RL д»Һ LLM иҝҒз§»еҲ° VLA з»қйқһз®ҖеҚ•зҡ„"жҚўдёӘжЁЎеһӢ"пјҢдёӨиҖ…еңЁзҠ¶жҖҒз©әй—ҙгҖҒеҠЁдҪңз©әй—ҙгҖҒзҺҜеўғеҸҚйҰҲе’ҢйҮҮж ·ж–№ејҸдёҠеӯҳеңЁж №жң¬жҖ§е·®ејӮгҖӮд»ҘдёӢйҖҗдёҖе®ҡд№үе№¶еҜ№жҜ”гҖӮ

2.1 LLM зҡ„ RL еҪўејҸеҢ–

зҠ¶жҖҒпјҲStateпјү$s_t$пјҡеңЁж—¶еҲ» $t$пјҢзҠ¶жҖҒз”ұиҫ“е…Ҙ prompt е’Ңе·Із”ҹжҲҗзҡ„ token еәҸеҲ—з»„жҲҗпјҡ

$$s_t = (x_{\text{prompt}}, y_1, y_2, \ldots, y_{t-1})$$

е…¶дёӯ $x_{\text{prompt}}$ жҳҜеҲқе§Ӣ promptпјҢ$y_t$ жҳҜз¬¬ $t$ дёӘз”ҹжҲҗзҡ„ tokenгҖӮе…ій”®зү№зӮ№пјҡзҠ¶жҖҒеҸӘеҢ…еҗ«ж–Үжң¬дҝЎжҒҜпјҢдё”е®Ңе…Ёз”ұжЁЎеһӢиҮӘиә«иҫ“еҮәй©ұеҠЁеўһй•ҝвҖ”вҖ”дёҚйңҖиҰҒеӨ–йғЁзҺҜеўғзҡ„е№Ійў„гҖӮ

еҠЁдҪңпјҲActionпјү$a_t$пјҡеҠЁдҪңжҳҜд»ҺиҜҚиЎЁ $\mathcal{V}$ дёӯйҖүжӢ©дёӢдёҖдёӘ tokenпјҡ

$$a_t = y_t \in \mathcal{V}, \quad y_t \sim \pi_\theta(\cdot|s_t) = \mathrm{softmax}\left(f_\theta(s_t) / T\right)$$

е…¶дёӯ $f_\theta(s_t) \in \mathbb{R}^{|\mathcal{V}|}$ жҳҜ LLM иҫ“еҮәзҡ„ logitsпјҢ$T$ жҳҜжё©еәҰеҸӮж•°пјҲжҺ§еҲ¶йҮҮж ·йҡҸжңәжҖ§пјүгҖӮе…ій”®зү№зӮ№пјҡеҠЁдҪңз©әй—ҙжҳҜзҰ»ж•Јзҡ„пјҲиҜҚиЎЁеӨ§е°ҸйҖҡеёё 32k-128kпјүпјҢдё”еҠЁдҪңзҡ„йҖүжӢ©йҖҡиҝҮ softmax + temperature sampling е®ҢжҲҗгҖӮ

зҺҜеўғпјҲEnvironmentпјүпјҡLLM зҡ„зҺҜеўғдёҚеҸӮдёҺдёӯй—ҙжӯҘйӘӨвҖ”вҖ”е®ғеңЁж•ҙдёӘеәҸеҲ—з”ҹжҲҗе®ҢжҲҗеҗҺжүҚз»ҷеҮәеҘ–еҠұдҝЎеҸ·гҖӮеңЁ rule-based RLпјҲеҰӮ DeepSeek-R1пјүдёӯпјҢеҘ–еҠұйҖҡеёёжҳҜеҹәдәҺжӯЈзЎ®жҖ§зҡ„дәҢе…ғеҲӨж–ӯпјҡ

$$r(\tau) = \begin{cases} 1, & \text{еҰӮжһң } \tau \text{ ж»Ўи¶іжӯЈзЎ®жҖ§ж ҮеҮҶ} \\ 0, & \text{еҗҰеҲҷ} \end{cases}, \quad \text{жҲ–} \quad r(\tau) = R_\phi(\tau) \in [0, 1]$$

е…¶дёӯ $R_\phi$ жҳҜеӯҰд№ еҲ°зҡ„еҘ–еҠұжЁЎеһӢпјҲеҰӮз”ЁдәҺ RLHF зҡ„ preference modelпјүпјҢ$\tau = (x_{\text{prompt}}, y_1, \ldots, y_{T_{\text{seq}}})$ жҳҜе®Ңж•ҙзҡ„з”ҹжҲҗеәҸеҲ—гҖӮе…ій”®зү№зӮ№пјҡеҘ–еҠұжҳҜ trajectory-level зҡ„вҖ”вҖ”еҸӘеңЁеәҸеҲ—з»“жқҹеҗҺз»ҷеҮәпјҢдёӯй—ҙжӯҘйӘӨжІЎжңүдҝЎеҸ·гҖӮиҝҷеӨ©з„¶йҖӮеҗҲ rule-based RLпјҲеҰӮ GRPOпјүпјҢеӣ дёәе®ғеҸӘе…іеҝғжңҖз»Ҳз»“жһңиҖҢйқһиҝҮзЁӢгҖӮ

RolloutпјҲйҮҮж ·пјүпјҡз»ҷе®ҡ prompt $x_{\text{prompt}}$пјҢLLM иҮӘеӣһеҪ’ең°йҖҗ token йҮҮж ·пјҢзӣҙеҲ°йҒҮеҲ° EOS token жҲ–иҫҫеҲ°жңҖеӨ§й•ҝеәҰгҖӮж•ҙдёӘиҝҮзЁӢжҳҜеҚ•еҗ‘зҡ„вҖ”вҖ”жЁЎеһӢиҫ“еҮә tokenпјҢtoken иҝҪеҠ еҲ°зҠ¶жҖҒдёӯпјҢеҫӘзҺҜеҫҖеӨҚзӣҙеҲ°з»ҲжӯўгҖӮжІЎжңүд»»дҪ•еӨ–йғЁзҺҜеўғеҸҚйҰҲд»Ӣе…ҘгҖӮйқһйӣ¶жё©еәҰ $T$ дҪҝзӯ–з•ҘеҸҜдә§з”ҹеӨҡж ·еҢ–зҡ„ rolloutsпјҢжҺўзҙўдёҚеҗҢи§Јйўҳи·Ҝеҫ„гҖӮ

зӣҙи§үзҗҶи§ЈпјҡLLM RL е°ұеғҸжҳҜ"й—ӯеҚ·зӯ”йўҳ"вҖ”вҖ”дҪ жӢҝеҲ°йўҳзӣ®пјҲpromptпјүпјҢеңЁи„‘еӯҗйҮҢжҖқиҖғе№¶еҶҷдёӢзӯ”жЎҲпјҲиҮӘеӣһеҪ’з”ҹжҲҗпјүпјҢжңҖеҗҺиҖҒеёҲеҸӘзңӢзӯ”жЎҲеҜ№дёҚеҜ№пјҲoutcome rewardпјүпјҢдёҚе…іеҝғдёӯй—ҙиҚүзЁҝеҶҷдәҶд»Җд№ҲгҖӮ

2.2 VLA зҡ„ RL еҪўејҸеҢ–

зҠ¶жҖҒпјҲStateпјү$s_t$пјҡзҠ¶жҖҒз”ұдёүзұ»еӨҡжЁЎжҖҒдҝЎжҒҜз»„жҲҗпјҡ

$$s_t = (o_t^{\text{vis}}, o_t^{\text{prop}}, l_{\text{task}})$$

  • $o_t^{\text{vis}}$пјҡеӨҡжЁЎжҖҒи§Ҷи§үи§ӮжөӢпјҲRGB еӣҫеғҸгҖҒж·ұеәҰеӣҫжҲ–зӮ№дә‘пјү
  • $o_t^{\text{prop}}$пјҡжң¬дҪ“ж„ҹзҹҘдҝЎжҒҜпјҲе…іиҠӮи§’еәҰгҖҒжң«з«Ҝжү§иЎҢеҷЁдҪҚе§ҝпјү
  • $l_{\text{task}}$пјҡд»»еҠЎиҜӯиЁҖжҢҮд»Ө

е…ій”®зү№зӮ№пјҡдёҺ LLM дёҚеҗҢпјҢVLA зҡ„зҠ¶жҖҒеҢ…еҗ«жқҘиҮӘзү©зҗҶдё–з•Ңзҡ„е®һж—¶ж„ҹзҹҘж•°жҚ®гҖӮи§Ҷи§үи§ӮжөӢжҜҸеҲ°ж–°ж—¶еҲ»йғҪдјҡеҸҳеҢ–пјҲеӣ дёәжңәжў°иҮӮз§»еҠЁдәҶгҖҒзү©дҪ“дҪҚзҪ®ж”№еҸҳдәҶпјүпјҢжң¬дҪ“ж„ҹзҹҘдҝЎжҒҜеҸҚжҳ жңәеҷЁдәәиҮӘиә«зҡ„зү©зҗҶзҠ¶жҖҒгҖӮиҝҷж„Ҹе‘ізқҖзҠ¶жҖҒдёҚжҳҜжЁЎеһӢиҮӘе·ұиғҪеҶіе®ҡзҡ„вҖ”вҖ”е®ғз”ұзҺҜеўғеҸҚйҰҲеҶіе®ҡгҖӮ

еҠЁдҪңпјҲActionпјү$a_t$пјҡеҠЁдҪңжҳҜжңәеҷЁдәәж“ҚдҪңз©әй—ҙдёӯзҡ„иҝһз»ӯжҺ§еҲ¶жҢҮд»ӨпјҢйҖҡеёёжҳҜжң«з«Ҝжү§иЎҢеҷЁеўһйҮҸжҲ–е…іиҠӮи§’еәҰзӣ®ж Үпјҡ

$$a_t = \text{Decoder}(h_\theta(s_t)), \quad \text{Decoder} \in \{\text{Diffusion Expert}, \text{Action Tokenizer}\}, \quad a_t \in \mathbb{R}^d$$

е…¶дёӯ $h_\theta(s_t)$ жҳҜ VLA жЁЎеһӢеҜ№зҠ¶жҖҒ $s_t$ зҡ„йҡҗи—ҸиЎЁзӨәпјҢDecoder жҳҜеҠЁдҪңи§Јз ҒеҷЁгҖӮ$d=7$ иЎЁзӨәе…ёеһӢзҡ„ 6-DoF дҪҚе§ҝ + еӨ№зҲӘдҪҚзҪ®гҖӮ

Decoder зҡ„дёӨзұ»е®һзҺ°пјҡ

  • Diffusion ExpertпјҲеҰӮ RDT-1BгҖҒПҖвӮҖпјүпјҡеңЁ latent space дёҠиҝӯд»ЈеҺ»еҷӘз”ҹжҲҗиҝһз»ӯеҠЁдҪңгҖӮиЎЁиҫҫиғҪеҠӣејәпјҢдҪҶдёҺ PPO зұ» RL з®—жі•е…је®№жҖ§е·®вҖ”вҖ”ж— жі•зӣҙжҺҘи®Ўз®— action зҡ„еҜ№ж•°жҰӮзҺҮгҖӮ
  • Action TokenizerпјҲеҰӮ OpenVLAгҖҒOpenVLA-OFTпјүпјҡе°Ҷиҝһз»ӯеҠЁдҪңз©әй—ҙзҰ»ж•ЈеҢ–дёәжңүйҷҗдёӘ binпјҢйҖҡиҝҮ LLM output head иҫ“еҮә action token зҡ„зҰ»ж•ЈеҲҶеёғгҖӮиҝҷжҳҜ SimpleVLA-RL йҖүжӢ©зҡ„ж–№ејҸпјҢеӣ дёәе®ғеӨ©з„¶жҸҗдҫӣ action distributionпјҢж”ҜжҢҒ PPO/GRPO жүҖйңҖзҡ„ random sampling е’Ң log-probability и®Ўз®—гҖӮ

е…ій”®зү№зӮ№пјҡеҠЁдҪңз©әй—ҙжҳҜиҝһз»ӯзҡ„дҪҶиў«зҰ»ж•ЈеҢ–еӨ„зҗҶгҖӮиҝҷи®© VLA еҸҜд»ҘеҖҹз”Ё LLM зҡ„иҫ“еҮәиҢғејҸпјҲsoftmax over tokensпјүпјҢеҗҢж—¶иҫ“еҮәзҡ„жҳҜзү©зҗҶдё–з•ҢдёӯеҸҜжү§иЎҢзҡ„иҝһз»ӯжҺ§еҲ¶жҢҮд»ӨгҖӮ

зҺҜеўғпјҲEnvironmentпјүпјҡзҺҜеўғжҳҜжңәеҷЁдәәж“ҚдҪңзҡ„зү©зҗҶдё–з•ҢжҲ–д»ҝзңҹеҷЁгҖӮе®ғжҸҗдҫӣзҠ¶жҖҒиҪ¬з§» $s_{t+1} = \text{Env}(s_t, a_t)$ е’ҢеҘ–еҠұдҝЎеҸ·пјҡ

$$r_t = \alpha \cdot \mathbb{I}_{\text{success}} + (1-\alpha) \cdot \sum_i w_i \cdot \phi_i(s_t, a_t), \quad \alpha \in [0,1]$$

$$\mathbb{I}_{\text{success}} = \begin{cases} 1, & \text{д»»еҠЎжҲҗеҠҹ} \\ 0, & \text{еҗҰеҲҷ} \end{cases}$$

е…¶дёӯ $\phi_i(s_t, a_t)$ жҳҜиҝҮзЁӢеҘ–еҠұпјҲеҰӮи·қзҰ»зӣ®ж Үзҡ„иҝңиҝ‘пјүпјҢ$w_i$ жҳҜеҗ„иҝҮзЁӢеҘ–еҠұзҡ„жқғйҮҚпјҢ$\alpha$ е№іиЎЎ outcome е’Ң process reward зҡ„жҜ”йҮҚгҖӮ

е…ій”®зү№зӮ№пјҡVLA зҡ„зҺҜеўғеҘ–еҠұеҸҜд»ҘжҳҜж··еҗҲзҡ„вҖ”вҖ”ж—ўжңүжңҖз»ҲжҲҗеҠҹдёҺеҗҰзҡ„дәҢе…ғдҝЎеҸ·пјҢд№ҹжңүдёӯй—ҙжӯҘйӘӨзҡ„зЁ еҜҶдҝЎеҸ·пјҲеҰӮи·қзҰ»зӣ®ж Үзҡ„зј©зҹӯпјүгҖӮдј з»ҹзҡ„жңәеҷЁдәә RL йҖҡеёёзІҫеҝғи®ҫи®ЎиҝҮзЁӢеҘ–еҠұ $\phi_i$пјҢдҪҶиҝҷдёҘйҮҚйҷҗеҲ¶дәҶеҸҜжү©еұ•жҖ§вҖ”вҖ”жҜҸдёӘд»»еҠЎйңҖиҰҒдё“еұһзҡ„еҘ–еҠұе·ҘзЁӢгҖӮSimpleVLA-RL зҡ„е…ій”®йҖүжӢ©жҳҜи®ҫ $\alpha = 1$пјҢеҚізәҜ outcome rewardпјҢд»ҺиҖҢйҒҝе…ҚиҝҷдёҖз“¶йўҲгҖӮ

RolloutпјҲйҮҮж ·пјүпјҡиҝҷжҳҜ LLM RL е’Ң VLA RL жңҖж №жң¬зҡ„еҢәеҲ«гҖӮVLA зҡ„ rollout жҳҜй—ӯзҺҜдәӨдә’йҮҮж ·пјҡ

  • жҜҸжӯҘпјҡзӯ–з•Ҙ $\pi_\theta$ жҺҘж”¶еҪ“еүҚзҠ¶жҖҒ $s_t$пјҢиҫ“еҮәдёҖдёӘ action chunk $(a_t, a_{t+1}, \ldots, a_{t+k-1})$пјҲй•ҝеәҰдёә $k$пјҢеҰӮ LIBERO дёӯ $k=8$пјҢRoboTwin дёӯ $k=25$пјү
  • жңәеҷЁдәәйЎәеәҸжү§иЎҢиҝҷ $k$ дёӘеҠЁдҪңпјҢзҺҜеўғж №жҚ®зү©зҗҶеҠЁеҠӣеӯҰдә§з”ҹжӣҙж–°зҠ¶жҖҒ
  • жү§иЎҢе®ҢжҜ•еҗҺпјҢжЁЎеһӢд»Ҙж–°зҠ¶жҖҒ $s_{t+k}$ дёәиҫ“е…ҘпјҢз”ҹжҲҗдёӢдёҖдёӘ action chunk
  • иҝҷдёҖиҝҮзЁӢжҢҒз»ӯеҲ°д»»еҠЎе®ҢжҲҗжҲ–иҫҫеҲ°жңҖеӨ§ episode й•ҝеәҰ

жңҖз»Ҳдә§з”ҹе®Ңж•ҙиҪЁиҝ№пјҡ$\tau = ((s_0, a_0), (s_1, a_1), \ldots, (s_T, a_T))$

зӣҙи§үзҗҶи§ЈпјҡVLA RL е°ұеғҸжҳҜ"й©ҫй©¶иҖғиҜ•"вҖ”вҖ”дҪ еқҗеңЁиҪҰйҮҢпјҢзңјзқӣзңӢи·ҜеҶөпјҲи§Ҷи§үи§ӮжөӢпјүпјҢжүӢжҸЎж–№еҗ‘зӣҳпјҲжң¬дҪ“ж„ҹзҹҘпјүпјҢиҖғе®ҳеңЁдҪ иҖіиҫ№иҜҙ"еүҚйқўи·ҜеҸЈе·ҰиҪ¬"пјҲиҜӯиЁҖжҢҮд»ӨпјүгҖӮдҪ жҜҸеҒҡдёҖдёӘж“ҚдҪңпјҲжү“ж–№еҗ‘зӣҳпјүпјҢиҪҰзҡ„дҪҚзҪ®е°ұеҸҳдәҶпјҲзҺҜеўғзҠ¶жҖҒиҪ¬з§»пјүпјҢдҪ йңҖиҰҒж №жҚ®ж–°зҡ„и·ҜеҶөеҒҡеҮәдёӢдёҖдёӘж“ҚдҪңгҖӮиҖғе®ҳеҸӘзңӢдҪ жңҖеҗҺжңүжІЎжңүеҲ°иҫҫзӣ®зҡ„ең°пјҲoutcome rewardпјүпјҢдёҚз®ЎдҪ дёӯйҖ”жҖҺд№ҲејҖзҡ„гҖӮиҝҷжҜ” LLM зҡ„"й—ӯеҚ·зӯ”йўҳ"еӨҚжқӮеҫ—еӨҡвҖ”вҖ”еӣ дёәжҜҸдёҖжӯҘзҡ„иҫ“е…ҘйғҪдҫқиө–дәҺд№ӢеүҚжүҖжңүжӯҘйӘӨзҡ„зү©зҗҶеҗҺжһңгҖӮ

2.3 LLM RL vs VLA RLпјҡж ёеҝғе·®ејӮжҖ»з»“

з»ҙеәҰ LLM RL VLA RL VLA еёҰжқҘзҡ„йўқеӨ–жҢ‘жҲҳ
State зәҜж–Үжң¬пјҢиҮӘеҢ…еҗ« еӨҡжЁЎжҖҒпјҢдҫқиө–зҺҜеўғ йңҖиҰҒе®һж—¶и§Ҷи§үзј–з ҒпјҢжҜҸжӯҘйғҪеҸҳ
Action зҰ»ж•Ј tokenпјҲ~128kпјү иҝһз»ӯвҶ’зҰ»ж•ЈеҢ–пјҲ256 binsпјү зҰ»ж•ЈеҢ–еј•е…ҘзІҫеәҰжҚҹеӨұ
Environment еҸӘеңЁз»“жқҹж—¶з»ҷеҸҚйҰҲ жҜҸжӯҘйғҪжңүзү©зҗҶдәӨдә’ ж…ўдё”жҳӮиҙөпјҲд»ҝзңҹжӯҘиҝӣ+жёІжҹ“пјү
Reward Rule-based / RM (0/1) еҸҜж··еҗҲ outcome + process жүӢе·ҘиҝҮзЁӢеҘ–еҠұдёҚеҸҜжү©еұ•
Rollout дёҖж¬ЎжҖ§иҮӘеӣһеҪ’з”ҹжҲҗ й—ӯзҺҜдәӨдә’йҮҮж · ж— жі•е№¶иЎҢз”ҹжҲҗеӨҡжқЎиҪЁиҝ№пјҲжҜҸжӯҘдҫқиө–дёҠдёҖжӯҘзҺҜеўғзҠ¶жҖҒпјү
йҮҮж ·йҖҹеәҰ жһҒеҝ«пјҲзәҜжҺЁзҗҶпјү ж…ўпјҲжҺЁзҗҶ + д»ҝзңҹ + жёІжҹ“пјү йңҖиҰҒеӨҡзҺҜеўғе№¶иЎҢ

зҗҶи§Јиҝҷдәӣе·®ејӮжҳҜзҗҶи§Ј SimpleVLA-RL жүҖжңүи®ҫи®ЎйҖүжӢ©зҡ„еүҚжҸҗпјҡдёәд»Җд№ҲйҖү token-based actionгҖҒдёәд»Җд№Ҳз”ЁзәҜ outcome rewardгҖҒдёәд»Җд№ҲйңҖиҰҒеӨҡзҺҜеўғе№¶иЎҢжёІжҹ“вҖ”вҖ”иҝҷдәӣйғҪеңЁеҗҺйқўзҡ„ж–№жі•йғЁеҲҶеҫ—еҲ°е…·дҪ“еӣһеә”гҖӮ

2.4 Group Relative Policy OptimizationпјҲGRPOпјү

GRPO жҳҜ DeepSeekMath жҸҗеҮәзҡ„ RL ж–№жі•пјҢд№ҹжҳҜ SimpleVLA-RL дҪҝз”Ёзҡ„ж ёеҝғз®—жі•гҖӮзҗҶи§Је®ғйңҖиҰҒе…ҲзҗҶи§Је®ғжӣҝжҺүдәҶд»Җд№ҲгҖӮ

дј з»ҹ PPO зҡ„й—®йўҳпјҡPPO йңҖиҰҒдёҖдёӘзӢ¬з«Ӣзҡ„ Value NetworkпјҲCriticпјүжқҘдј°и®ЎжҜҸдёӘзҠ¶жҖҒзҡ„д»·еҖј $V(s)$пјҢз”ЁдәҺи®Ўз®— advantage $A_t = R_t - V(s_t)$гҖӮиҝҷж„Ҹе‘ізқҖпјҡ

  • йўқеӨ–и®ӯз»ғдёҖдёӘдёҺ Policy и§„жЁЎзӣёеҪ“зҡ„зҘһз»ҸзҪ‘з»ңпјҲ2Г— жҳҫеӯҳпјү
  • Critic иҮӘиә«д№ҹжңүдј°и®ЎиҜҜе·®пјҢзү№еҲ«жҳҜеңЁй•ҝзЁӢд»»еҠЎдёӯиҜҜе·®зҙҜз§ҜдёҘйҮҚ
  • еңЁ VLA еңәжҷҜдёӯпјҢCritic йңҖиҰҒзҗҶи§ЈеӨҡжЁЎжҖҒзҠ¶жҖҒвҖ”вҖ”иҝҷжң¬иә«е°ұжҳҜдёҖдёӘйҡҫйўҳ

GRPO зҡ„ж ёеҝғжҖқжғіпјҡз”ЁдёҖдёӘ prompt зҡ„еӨҡжқЎйҮҮж ·иҪЁиҝ№д№Ӣй—ҙзҡ„зӣёеҜ№жҜ”иҫғпјҢжӣҝд»Ј Value Network зҡ„з»қеҜ№дј°и®ЎгҖӮ

е…·дҪ“еҒҡжі•пјҡеҜ№еҗҢдёҖдёӘеҲқе§ӢзҠ¶жҖҒ $s_0$пјҲеҚіеҗҢдёҖдёӘ task promptпјүпјҢbehavior policy $\pi_{\theta_{\text{old}}}$ з”ҹжҲҗ $G$ жқЎиҪЁиҝ№ $\{\tau_i\}_{i=1}^G$пјҲжң¬ж–Үдёӯ $G=8$пјүгҖӮGRPO зҡ„зӣ®ж ҮеҮҪж•°йҮҮз”Ё PPO-style clippingпјҢй…ҚеҗҲ KL жӯЈеҲҷеҢ–зәҰжқҹзӯ–з•Ҙжӣҙж–°пјҡ

$$J_{\text{GRPO}}(\theta) = \mathbb{E}_{s_0 \sim \mathcal{D}, \{\tau_i\} \sim \pi_{\theta_{\text{old}}}} \Bigg[ \frac{1}{G} \sum_{i=1}^G \frac{1}{|\tau_i|} \sum_{t=1}^{|\tau_i|} \min\left( r_{i,t}(\theta) \hat{A}_i, \text{clip}(r_{i,t}(\theta), 1-\epsilon, 1+\epsilon) \hat{A}_i \right) - \beta D_{\text{KL}}(\pi_\theta || \pi_{\text{ref}}) \Bigg]$$

е…¶дёӯдёӨдёӘе…ій”®йҮҸпјҡ

Importance Sampling Ratio $r_{i,t}(\theta)$пјҡ

$$r_{i,t}(\theta) = \frac{\pi_\theta(a_{i,t}|s_{i,t})}{\pi_{\theta_{\text{old}}}(a_{i,t}|s_{i,t})}$$

иҝҷжҳҜж–°ж—§зӯ–з•ҘеңЁеҗҢдёҖзҠ¶жҖҒ-еҠЁдҪңеҜ№дёҠзҡ„жҰӮзҺҮд№ӢжҜ”гҖӮ$r > 1$ иЎЁзӨәж–°зӯ–з•ҘжҜ”ж—§зӯ–з•ҘжӣҙеҒҸеҘҪиҜҘеҠЁдҪңпјҢ$r < 1$ еҲҷзӣёеҸҚгҖӮPPO зҡ„ clipping е°ҶжӯӨ ratio йҷҗеҲ¶еңЁ $[1-\epsilon, 1+\epsilon]$ иҢғеӣҙеҶ…пјҢйҳІжӯўеҚ•жӯҘжӣҙж–°иҝҮеӨ§еҜјиҮҙи®ӯз»ғдёҚзЁіе®ҡгҖӮ

Group-Normalized Advantage $\hat{A}_i$пјҡ

$$\hat{A}_i = \frac{R_i - \text{mean}(\{R_i\}_{i=1}^G)}{\text{std}(\{R_i\}_{i=1}^G)}$$

иҝҷжҳҜ GRPO зҡ„ж ёеҝғеҲӣж–°гҖӮ$R_i$ жҳҜз¬¬ $i$ жқЎиҪЁиҝ№зҡ„жҖ»еҘ–еҠұпјҲеңЁ SimpleVLA-RL дёӯе°ұжҳҜ 0 жҲ– 1пјүгҖӮйҖҡиҝҮз»„еҶ… z-score ж ҮеҮҶеҢ–пјҡ

  • жҲҗеҠҹиҪЁиҝ№пјҲ$R_i=1$пјүиҺ·еҫ—жӯЈзҡ„ advantageпјҡиў«зӯ–з•ҘејәеҢ–
  • еӨұиҙҘиҪЁиҝ№пјҲ$R_i=0$пјүиҺ·еҫ—иҙҹзҡ„ advantageпјҡиў«зӯ–з•ҘжҠ‘еҲ¶
  • еҰӮжһңе…Ёз»„ 8 жқЎйғҪжҲҗеҠҹжҲ–йғҪеӨұиҙҘпјҢ$\text{std}=0$пјҢжүҖжңү $\hat{A}_i = 0$пјҢжўҜеәҰж¶ҲеӨұвҖ”вҖ”иҝҷи§ЈйҮҠдәҶдёәд»Җд№ҲйңҖиҰҒ Dynamic Sampling

еӯҗеӣҫ (a) Dynamic SamplingпјҲеҠЁжҖҒйҮҮж ·иҝҮж»Өпјү

иҰҒи§ЈеҶізҡ„й—®йўҳпјҡжўҜеәҰж¶ҲеӨұгҖӮGRPO зҡ„дјҳеҠҝдј°и®Ўе®Ңе…Ёдҫқиө–з»„еҶ…зӣёеҜ№жҜ”иҫғпјҡ

$$\hat{A}_i = \frac{R_i - \text{mean}(\{R_i\}_{i=1}^G)}{\text{std}(\{R_i\}_{i=1}^G)}$$

еҪ“еҗҢдёҖ prompt зҡ„ $G=8$ жқЎиҪЁиҝ№е…ЁйғЁжҲҗеҠҹпјҲ$R_i$ е…Ёдёә 1пјүжҲ–е…ЁйғЁеӨұиҙҘпјҲ$R_i$ е…Ёдёә 0пјүж—¶пјҢ$\text{std}=0$пјҢжүҖжңүиҪЁиҝ№зҡ„дјҳеҠҝ $\hat{A}_i = 0$гҖӮжӯӨж—¶жўҜеәҰе®Ңе…Ёж¶ҲеӨұпјҢиҝҷдёҖжү№ж•°жҚ®еҜ№зӯ–з•Ҙжӣҙж–°жІЎжңүд»»дҪ•иҙЎзҢ®вҖ”вҖ”зӣёеҪ“дәҺзҷҪеҒҡдәҶ 8 жқЎиҪЁиҝ№зҡ„йҮҮж ·е’ҢжҺЁзҗҶгҖӮеңЁ RL и®ӯз»ғж—©жңҹпјҢз”ұдәҺзӯ–з•ҘиғҪеҠӣдёҚи¶іпјҢе…ЁеӨұиҙҘз»„йў‘з№ҒеҮәзҺ°пјӣеҲ°дәҶи®ӯз»ғеҗҺжңҹпјҢйҡҸзқҖзӯ–з•ҘжҸҗеҚҮпјҢе…ЁжҲҗеҠҹз»„еҸҲйў‘з№ҒеҮәзҺ°гҖӮж— и®әе“Әз§Қжғ…еҶөпјҢйғҪеҜјиҮҙеӨ§йҮҸи®ӯз»ғжӯҘзҡ„жўҜеәҰдҝЎеҸ·дёәйӣ¶пјҢи®ӯз»ғжһҒдёҚзЁіе®ҡгҖӮ

жҖҺд№ҲеҒҡпјҡеңЁжҜҸж¬Ў rollout йҮҮж ·еҗҺпјҢжЈҖжҹҘжҜҸз»„зҡ„ 8 жқЎиҪЁиҝ№ outcomeгҖӮеҰӮжһңдёҖз»„е…ЁжҲҗеҠҹжҲ–е…ЁеӨұиҙҘпјҢзӣҙжҺҘдёўејғиҜҘз»„пјҢйҮҚж–°йҮҮж ·пјҢзӣҙеҲ°ж”¶йӣҶеҲ°и¶іеӨҹж•°йҮҸпјҲbatch_sizeпјүзҡ„"ж··еҗҲз»“жһңз»„"вҖ”вҖ”еҚіжҜҸз»„еҶ…иҮіе°‘жңүдёҖжқЎжҲҗеҠҹе’ҢиҮіе°‘жңүдёҖжқЎеӨұиҙҘгҖӮеҪўејҸеҢ–зәҰжқҹжқЎд»¶пјҡ

$$0 < \left| \{\text{traj}_i \mid \text{is\_successful}[\text{traj}_i]\} \right| < G$$

д»Јз Ғе®һзҺ°пјҲray_trainer.py:filter()пјүпјҡиҝҮж»Ө accuracy е…Ё 0 жҲ–е…Ё 1 зҡ„ groupпјҢacc_mask = (acc_tensor >= lower) & (acc_tensor <= upper)пјҢеҸӘдҝқз•ҷж··еҗҲз»“жһңз»„гҖӮ

ж•Ҳжһңпјҡеӣҫдёӯи“қиүІжӣІзәҝпјҲжңӘеҗҜз”ЁпјүжіўеҠЁеү§зғҲпјҢи®ӯз»ғдёҚзЁіе®ҡпјҢжңҖз»Ҳж”¶ж•ӣиҮізәҰ 95%гҖӮзәўиүІжӣІзәҝпјҲеҗҜз”Ё Dynamic SamplingпјүиҝҮж»ӨдәҶж— жўҜеәҰдҝЎеҸ·зҡ„ж— ж•Ҳз»„пјҢзЎ®дҝқжҜҸдёҖжӯҘжӣҙж–°йғҪжңүйқһйӣ¶дјҳеҠҝдј°и®ЎпјҢи®ӯз»ғжӣІзәҝе№іж»‘зЁіе®ҡпјҢжңҖз»Ҳж”¶ж•ӣиҮізәҰ 98%гҖӮиҝҷдёҖж–№жі•еңЁ LLM RL дёӯе·Іиў« DAPOгҖҒKimi-K1.5 зӯүе·ҘдҪңйӘҢиҜҒжңүж•ҲпјҢSimpleVLA-RL е°Ҷе…¶еј•е…Ҙ VLA RL еңәжҷҜгҖӮ

еӯҗеӣҫ (b) Clip HigherпјҲйқһеҜ№з§°иЈҒеүӘдёҠз•Ңпјү

иҰҒи§ЈеҶізҡ„й—®йўҳпјҡдҪҺжҰӮзҺҮ token зҡ„жҺўзҙўеҸ—йҷҗгҖӮPPO/GRPO зҡ„ж ёеҝғж“ҚдҪңжҳҜеҜ№ importance sampling ratio $r_{i,t}(\theta) = \pi_\theta(a_{i,t}|s_{i,t}) / \pi_{\theta_{\text{old}}}(a_{i,t}|s_{i,t})$ иҝӣиЎҢиЈҒеүӘпјҲclippingпјүпјҢйҳІжӯўеҚ•жӯҘзӯ–з•Ҙжӣҙж–°иҝҮеӨ§пјҡ

$$\text{clip}(r, 1-\varepsilon, 1+\varepsilon)$$

жҖҺд№ҲеҒҡпјҡеҖҹйүҙ DAPOпјҢе°ҶиЈҒеүӘиҢғеӣҙж”№дёәйқһеҜ№з§°зҡ„ $[1-\varepsilon_{low}, 1+\varepsilon_{high}] = [0.8, 1.28]$гҖӮдёӢз•ҢдҝқжҢҒ $0.8$ дёҚеҸҳпјҲйҳІжӯўйҒ—еҝҳеҘҪеҠЁдҪңпјүпјҢдёҠз•Ңд»Һ $1.2$ жҸҗй«ҳеҲ° $1.28$гҖӮиҝҷж„Ҹе‘ізқҖж–°зӯ–з•ҘеҸҜд»Ҙе°ҶдҪҺжҰӮзҺҮ token зҡ„жҰӮзҺҮжҸҗеҚҮеҲ°ж—§зӯ–з•Ҙзҡ„ $1.28$ еҖҚпјҢз»ҷдәҲжҺўзҙўжӣҙеӨ§зҡ„з©әй—ҙгҖӮ

  • $T>1.0$пјҡеҲҶеёғиў«"жӢүе№і"пјҢй«ҳжҰӮзҺҮе’ҢдҪҺжҰӮзҺҮ token д№Ӣй—ҙзҡ„е·®и·қзј©е°ҸпјҢдҪҺжҰӮзҺҮ token иў«йҮҮеҲ°зҡ„жңәдјҡеўһеҠ

def compute_policy_loss(old_log_prob, log_prob, advantages, eos_mask,
                        clip_ratio_high, clip_ratio_low):
    ratio = torch.exp(log_prob - old_log_prob)  # importance sampling ratio
    pg_losses = -advantages * ratio
    pg_losses2 = -advantages * torch.clamp(ratio, 1-clip_ratio_low, 1+clip_ratio_high)
